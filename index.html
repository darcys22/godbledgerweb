GoDBLedger
The Open Source Accounting System With Features That Programmers Expect

Usage

GoDBLedger is a central server for all your financial transactions. It acts as the main endpoint for all your financial apps to talk to and it safely stores all the transactions in a database of your choice. Out of the box it functions with SQLite and MySQL.

Database Backends - SQL

--Trial Balance directly from MySQL
SELECT
  split_accounts.account_id as Account,
  SUM(splits.amount) as Balance
FROM splits
JOIN split_accounts
  ON splits.split_id = split_accounts.split_id
WHERE splits.split_date <= CURDATE()
GROUP BY split_accounts.account_id
